📊 CommercePulse Analytics Dashboard

## 📝 Project Overview
This project presents an interactive Power BI dashboard designed to analyze e-commerce sales performance, profitability, and business trends.  
The dashboard provides a comprehensive view of key metrics and helps in identifying revenue drivers, loss-making areas, and the impact of discounts on profitability.

---

## 🎯 Objectives
- Analyze overall sales and profit performance
- Identify top-performing and underperforming sub-categories
- Understand the relationship between sales, profit, and discounts
- Provide actionable business insights through interactive visuals

---

## 📌 Key Features
- 🔹 Sales & Profit Analysis
- 🔹 Sub-category level performance insights
- 🔹 Discount vs Profit relationship analysis
- 🔹 Interactive filters (Category, Segment, Ship Mode)
- 🔹 Clean and minimal dashboard design
- 🔹 Advanced visuals like Scatter Plot, Treemap, and Waterfall Chart

---

## 📊 Dashboard Pages

### 🔹 Page 1: Overview Dashboard
- KPI Cards (Total Sales, Profit, Orders, Profit Margin, Avg Order Value)
- Sales Trend Over Time (Line Chart)
- Top 5 Sub-Categories by Sales (Bar Chart)
- Sales vs Profit by Sub-Category (Scatter Plot)
- Product-level performance table with conditional formatting

---

### 🔹 Page 2: Profit & Performance Insights
- Sales Distribution by Category (Treemap)
- Profit Contribution Analysis (Waterfall Chart)
- Interactive filters for deep analysis
- Focus on profitability drivers and loss-making segments

---

## 🛠 Tools & Technologies Used
- Power BI
- DAX (Data Analysis Expressions)
- Data Modeling
- Data Visualization Techniques

---

## 📂 Dataset
- E-commerce Superstore Dataset
- Includes fields like:
  - Sales
  - Profit
  - Discount
  - Category & Sub-Category
  - Customer Segment
  - Ship Mode
 
   📈 Key Insights
- Technology category contributes the highest sales
- Some sub-categories negatively impact overall profitability
- Higher discounts often lead to reduced profit margins
- Profitability varies significantly across different segments and shipping modes
